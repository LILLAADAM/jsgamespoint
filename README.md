# JSGamesPoint

A modern, responsive gaming marketplace website built with Next.js 15, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Create production build
- `npm run start` — Serve production build
- `npm run lint` — Run ESLint

## Content

All site copy and data lives in the `data/` folder. Edit those files to update text across the site.

## Deployment

Deploy to [Vercel](https://vercel.com), Netlify, or any platform that supports Next.js. Run `npm run build` then `npm run start`.
